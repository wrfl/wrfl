## How to Run

1. `bundle`
1. `bundle exec rake db:migrate db:seed`
1. `rails s`
1. Visit [http://localhost:3000](http://localhost:3000).
1. Log in as `test@example.com` / `password` for a staff user, or `admin@example.com` / `password` for an admin user.
